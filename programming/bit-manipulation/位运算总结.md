# 位运算总结

1. n & (n-1) : 可以直接去除n的最后一个1
示例题目: 191 计算位1的个数
2. n & ~(1 << k): 置第i位为0

3. 提取奇数和偶数位
奇数位全1 -> 0101.... 表示为 0x55555555
偶数位全1 -> 1010.... 表示为 0xaaaaaaaa


4. 判断奇偶：
奇：(x & 1) == 1 
偶：(x & 1) == 0 
https://leetcode.cn/problems/exchange-lcci/description/?envType=study-plan-v2&envId=cracking-the-coding-interview

1. 去除最后一位 1：x & (x - 1)

2. 得到最后一位 1：x & -x

3. 判断 2 的幂次：x & (x - 1) == 0

4. 交换两个数：a ^= b; b ^= a; a ^= b;

5. 将最左边的 n 位清零：x & (~0 << n)

6. 获取 x 的第 n 位值（0 或 1）：(x >> n) & 1

7.  获取 x 的第 n 位的幂值：x & (1 << n)

仅将第 n 位置为 1：x | (1 << n)

仅将第 n 位置为 0：x & (~(1 << n))

将 x 最高位至第 n 位（含）清零：x & ((1 << n) - 1)

将第 n 位至第 0 位（含）清零：x & (~((1 << (n + 1)) - 1))

取反第 n 位：x ^ (1 << n)

异或满足交换律、结合律：a ^ b ^ a = b ^ (a ^ a) = b ^ 0 = b
