# 完全二叉树

- 完全二叉树: 除了最后一层外，每一层都是满的，最后一层节点靠左
- 满二叉树: 所有层都是满的, 节点个数: $2^h-1$

## 1. n个节点的完全二叉树,求叶子节点的个数
度: 一个节点的度为其子节点的个数，因此在二叉树中度为n0,n1,n2三种情况
$$
\begin{aligned}
    n_0 + n_1 + n_2 = n \\
    n_1 + 2*n_2 = n - 1 \\
    所以: n0 = n2 +1; \\
    n0 = (n - n1 +1) / 2
\end{aligned}
$$
注意思考: 在二叉树中，n1 = 1或者0
所以即可求出叶子节点的个数了
